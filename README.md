IMyCloneable<T>
	Плюсы: типобезопасность, удобство использования, современный подход
	Минусы: не является частью .NET Framework, ограниченная совместимость, избыточность реализации
	
ICloneable
	Плюсы: является частью .NET Framework, простота базовой реализации, полиморфизм
	Минусы: отсутствие типобезопасности, невозможность сообщить какой тип копирования реализован, нарушение принципа Б.Лисков
